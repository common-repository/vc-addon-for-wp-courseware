This plugin adds integration between WPBakery Page Builder and WP Courseware to customize unit content with WPBakery Page Builder elements.

== Description ==
[Fly Plugins](https://flyplugins.com) presents [WP Courseware](https://flyplugins.com/wp-courseware) for [WPBakery Page Builder](https://wpbakery.com/).

= Do you want to use the power of WPBakery Page Builder and create custom course units? =
The WP Courseware integration is designed to give you control of creating custom layouts for your course units with WPBakery Page Builder.

== Installation ==

1. Upload the `WPBakery Page Builder Addon for WP Courseware` folder into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
